A simple Flask API to locate cafes with WiFi and power sockets for remote working.

You can use this API to get information about cafes, add new cafes, update cafe details, and delete cafes from the database.

Available Endpoints:

  • GET /random - Get a random cafe
  • GET /all - Get all cafes
  • GET /search?loc=LOCATION - Search cafes by location
  • POST /add - Add a new cafe (requires form data)
  • PATCH /update-price/ID?new_price=PRICE - Update cafe price
  • DELETE /report-closed/ID?api-key=KEY - Delete a cafe (requires API key)
